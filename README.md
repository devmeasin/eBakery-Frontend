## Run Locally

Clone the project

```bash
  git clone https://github.com/easinwebpro/
```

Go to the project directory

```bash
  cd eBakery-Frontend
```

Install dependencies

```bash
  yarn or npm i
```

Start the server

```bash
  npm run start
```

**Open [http://localhost:3000](http://localhost:3000) to view it in your browser.**
